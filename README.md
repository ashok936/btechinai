# BtechInAI

**BtechInAI** is a dedicated educational platform for Artificial Intelligence students, offering high-quality courses, career guidance, and hands-on projects. This Jekyll-powered website serves as a central hub for AI aspirants to enhance their knowledge and skills.

## Features

- **Premium AI Courses**: Structured curriculum covering Machine Learning, Deep Learning, and Neural Networks.
- **Career Mentorship**: Strategic guidance to help students secure top roles in the AI industry.
- **Hands-on Projects**: Real-world applications and R&D opportunities to build a strong portfolio.
- **Blog**: Latest updates, insights, and articles on Artificial Intelligence.

## Getting Started

### Prerequisites

- **Ruby** (2.6.10 recommended)
- **Bundler**

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd btechinai
    ```

2.  Install dependencies:
    ```bash
    bundle install --path vendor/bundle
    ```

### Development

Run the Jekyll server locally:

```bash
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000`.

### Deployment

This site is configured for deployment on **GitHub Pages**. Ensure you have a `CNAME` file in the root directory pointing to `btechinai.com`.
