## Syllabus Details
{: .section-title }

### AI 201 - Introduction to Artificial Intelligence

**Key Topics:**
- AI history and fundamentals
- Intelligent agents and environments
- Uninformed and informed search
- Knowledge representation
- Expert systems

**Study Tips:**
- Implement search algorithms from scratch
- Study AI applications in industry
- Practice problem formulation
- Review foundational AI concepts regularly

---

### AI 202 - Machine Learning Fundamentals

**Key Topics:**
- Supervised learning algorithms
- Unsupervised learning and clustering
- Regression and classification
- Model evaluation and validation
- Hyperparameter tuning

**Study Tips:**
- Implement ML algorithms using Python
- Practice with real datasets from Kaggle
- Understand bias-variance tradeoff
- Build 3-5 end-to-end projects

---

### MTH 201 - Probability and Statistics

**Key Topics:**
- Probability theory and distributions
- Descriptive and inferential statistics
- Hypothesis testing methods
- Regression analysis
- Correlation and covariance

**Study Tips:**
- Master foundational probability concepts
- Practice statistical tests and calculations
- Connect theory to ML applications
- Analyze real datasets with statistical methods

---

### CSC 201 - Database Management Systems

**Key Topics:**
- Relational database design
- SQL queries and optimization
- Normalization and ACID properties
- Transactions and concurrency
- Indexing strategies

**Study Tips:**
- Write complex SQL queries regularly
- Understand database design patterns
- Practice query optimization
- Study real database systems (PostgreSQL, MySQL)

---

### CSC 202 - Computer Networks

**Key Topics:**
- OSI and TCP/IP models
- IP addressing and routing
- Transport layer protocols (TCP, UDP)
- Network security basics
- DNS and HTTP protocols

**Study Tips:**
- Understand packet-level communication
- Use tools like Wireshark for analysis
- Study network protocols in depth
- Practice network troubleshooting

---

### CSC 203 - Software Engineering

**Key Topics:**
- Software development lifecycle
- Requirements engineering
- Design principles and patterns
- Testing strategies
- Software maintenance and evolution

**Study Tips:**
- Participate in team programming projects
- Study real software architectures
- Practice design pattern implementation
- Learn version control workflows
