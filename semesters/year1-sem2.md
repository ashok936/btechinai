## Syllabus Details
{: .section-title }

### CMP 102 - Object Oriented Programming

**Key Topics:**
- Classes, objects, and methods
- Inheritance and polymorphism
- Encapsulation and abstraction
- Design patterns fundamentals
- Exception handling

**Study Tips:**
- Practice designing class hierarchies
- Understand SOLID principles early
- Code multiple OOP projects
- Review real-world design patterns

---

### CSC 102 - Data Structures and Algorithms

**Key Topics:**
- Arrays, linked lists, stacks, queues
- Trees and tree traversal
- Graphs and graph algorithms
- Sorting and searching techniques
- Algorithm complexity analysis

**Study Tips:**
- Visualize data structure operations
- Practice complexity calculations
- Implement each structure from scratch
- Solve 20+ algorithm problems weekly

---

### MTH 102 - Mathematics II (Linear Algebra)

**Key Topics:**
- Matrix operations and determinants
- Linear systems and Gaussian elimination
- Eigenvalues and eigenvectors
- Vector spaces and basis
- Applications in computer graphics

**Study Tips:**
- Connect linear algebra to AI/ML applications
- Solve matrix problems by hand first
- Use computational tools for verification
- Study geometric interpretations

---

### PHY 102 - Physics II

**Key Topics:**
- Electrostatics and electromagnetism
- Magnetic fields and induction
- Waves and interference
- Optics and light properties
- Modern physics fundamentals

**Study Tips:**
- Draw force diagrams for every problem
- Perform hands-on lab experiments
- Connect concepts to real applications
- Lab reports contribute 20% to grade

---

### MTH 103 - Discrete Mathematics

**Key Topics:**
- Propositional and predicate logic
- Set theory and relations
- Functions and combinatorics
- Graph theory fundamentals
- Proof techniques and mathematical induction

**Study Tips:**
- Practice logic proofs systematically
- Visualize graph structures
- Understand counting principles deeply
- Solve combinatorics problems daily

---

### CSC 103 - Microprocessor and Assembly Language

**Key Topics:**
- Microprocessor architecture
- Assembly programming basics
- Memory management and registers
- Interrupt handling
- I/O operations

**Study Tips:**
- Write simple assembly programs
- Understand CPU instruction sets
- Practice addressing modes
- Lab work is heavily graded
